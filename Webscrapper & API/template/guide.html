API user Guide

Thank you for using our API

Throughout this project we wanted to design a webscrapping tool in order to retrieve macroeconomic indicators and news from both TradingEconomics.com and ForexFactory.com. We used the Beautifull Soup and Selenium python packages in order to do so and tried to generalize the scrapping code as much as possible so that the user can quickly add any additional types of macroeconomic variables. Please add a new "key" that corresponds to the last part of the URL website from the desired indicator on TradingEconomics in the list of keys inside of the "Scrapping" Ipynb File.

We also created this API to quickly display the various macroeconomic data frames and recent Macro news announcements. We used the Flask Package to create this website and some CSS templates from W3 Schools. We also propose a Dashboard made with the Dash library which is based on Flask. We can integrate this Dashboard within the Flask API using the following methodology: https://hackersandslackers.com/plotly-dash-with-flask/

To navigate through the website, users have access to the news directly on the landing/home page.

Users can have access to the different macroeconomic data for various countries by clicking on the links/buttons at the top of the website.

Please refer to the various Jupyter Notebooks for more details.
